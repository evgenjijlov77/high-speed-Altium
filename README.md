# high-speed-Altium
Практическое пособие по высокочастотной трассировке и использованию Altium Designer, нацеленное на студентов и новичков. 
ВНИМАНИЕ: В тексте встречается сленг и ненормативная лексика, форма - свободная, не академическая. 
### Содержание:
Интерфейс Altium Designer (на основе 24 версии) - schematic и PCB редакторы
Разбор базовых понятий
Импеданс и нюансы работы с ним
Материалы печатных плат, производство и как с этим работать
Согласование сигналов по временной задержке
Разбор конкретных интерфейсов (приведены те, с которыми я лично работал)
### Цель проекта:
Распространение полностью свободное, цель - поделиться знанием, собрать обратную связь для улучшения собственной модели и текстового пособия.

Также в репозитории присутствует файл программы для расчета задержки в переходном отверстии платы. Разработанно моим коллегой, распространяется с его согласия.
### Немного об авторе:
Разработчик печатных плат, опыт работы в коммерческой разработке - 2 года (до этого максимум мог посредственно развести микроконтроллер). Из значимых успешных проектов - плата PCIe хаба (m2 - шлейф - плата с микросхемой хаба и разъемами x1), материнская плата на RK3588 (LPDDR4X-4266MT/s, HDMI2.1, PCIe2/3, Ethernet 2.5G, USB2/3, SATA3), UDIMM DDR4 3200 MT/s. Образование - бакалавр, радиофизика. Профессиональные навыки формировались самостоятельно (документация и формирование моделей в голове), поэтому могут быть упрощения, неточности, небольшие ошибки. Работа осуществлялась только с российскими производителями печатных плат (как модификатор сложности), поэтому HDI-инструменты описаны посредственно.
