# high-speed-Altium
Практическое пособие по высокочастотной трассировке и использованию Altuim Designer, нацеленное на студентов и новичков. В тексте встречается сленг и ненормативная лексика, форма - свободная, не академическая. 
Содержание:
Интерфейс Altium Designer (на основе 24 версии) - schematic и PCB редакторы
Разбор базовых понятий
Импеданс и нюансы работы с ним
Материалы печатных плат, производство и как с этим работать
Согласование сигналов по временной задержке
Разбор конкретных интерфейсов (приведены те, с которыми я лично работал)
Распространение полностью свободное, цель - поделиться знанием, собрать обратную связь для улучшения собственной модели и текстового пособия. 
Почта для обратной связи и вопросов: evgenjijlov77@gmail.com
Настоятельная рекомендация к просмотру: https://www.youtube.com/watch?v=c-VAPqNBDRU&t=5407s
Данное видео создано не мной, это лишь рекомендация. Для меня оно стало "точкой входа" в профессиональную трассировку.
Также в репозитории присутствует файл программы для расчета задержки в переходном отверстии платы. Разработанно моим коллегой, распространяется с его согласия.
